# PokeExplorer

Catálogo interativo de Pokémon desenvolvido como Trabalho Prático para a disciplina de Programação e Design para Web II da FAETERJ - Unidade Barra Mansa.

O objetivo da aplicação é demonstrar o domínio prático dos conceitos fundamentais do Next.js, incluindo o uso de App Router, rotas dinâmicas, componentização e consumo da PokéAPI pública (via requisições GET).

## 🚀 Tecnologias Utilizadas

- **[Next.js](https://nextjs.org/)** (App Router - Versão 14+)
- **React**
- **TypeScript**
- **CSS (Global Styles & Soft UI)**
- **[PokéAPI](https://pokeapi.co/)** (Consumo de dados e imagens oficiais)

## 📋 Pré-requisitos

Antes de começar, certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

## 🛠️ Como executar o projeto localmente

Siga os passos abaixo para clonar e rodar a aplicação na sua máquina:

1. **Clone o repositório**
   ```bash
   git clone [https://github.com/FernandoRCoelho5/pokeexplorer.git](https://github.com/FernandoRCoelho5/pokeexplorer.git)

2. **Acesse a pasta do projeto**
   ```bash
   cd pokeexplorer

4. **Instale as Dependências**
   ```bash
    npm install

5. **Inicie o servidor de desenvolvimento**
   ```bash
    npm run dev

6. **Acesse a Aplicação**
    Acesse http://localhost:3000 no navegador
